# AI_for_Business_Laboratory
This repo is a help for students on Industrial Engineer to understand about AI basics. Since basic hypothesis test until 2^k designs. I used CLAUDE to make the code. I use the theory concept of Montgomery books about DOE. I make the activities of labs with a new problems of my own.

- LAB_IA_3 : EDA 2 / Pre Processing (Tidy Data), EDA, AND SKLearn (Test Set and Validation Set, Linear Regression, Metrics)
- LAB_IA_4 : Data Visualization (MatplotLib, Seaborn, Ploty)
