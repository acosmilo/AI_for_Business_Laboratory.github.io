Cómo mantener el menú

  1. Deja este archivo como index.html en la raíz del repositorio.
  2. Activa GitHub Pages desde Settings, Pages, branch main, carpeta /root.
  3. Cuando agregues una clase nueva, copia una tarjeta de laboratorio y cambia el número, el título y la ruta del HTML.
  4. Respeta mayúsculas, guiones bajos y nombres de carpetas. GitHub Pages no perdona rutas mal escritas, porque claro, alguien tenía que ser estricto.
